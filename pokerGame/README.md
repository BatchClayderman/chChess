## pokerGame

该文件夹提供扑克牌游戏可能的实现。

### pokerGame.cpp

这是一个你能打赢，同时支持多种扑克牌类型的 C++ 程序，但暂未完善。

该代码支持命令行启动，在 Windows 下应当使用 Visual Studio 进行编译，在其它平台下可使用 g++ 直接编译。

前有代码即证明（Code as proof），现有代码即规则（Code as rules）。
